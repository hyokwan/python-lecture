
[기본 데이터 탐색]
1. 타겟폴더경로를 지정한다.
2. 폴더내에 .csv파일을 전부 수집해서 리스트에 넣는다

[데이터 정합성검증]
- 데이터의 row를 카운트한다.
- 중복 데이터의 row를 카운트 한다.
- 검증로직
   * 두개 카운트 개수가 같은경우 PASS
   * 두개 카운트 개수가 다른경우 FAIL

[결과데이터프레임]
- 파일명: 탐색폴더 정의된 파일명
- 전체ROW개수: 불러온 데이터의 전체 행 개수 (.shape[0])
- 전체중복ROW개수: 중복제거 후 전체 행 개수 (.drop_du->shap[0]_
- 최종결과: 개수 동일한경우 "PASS" 동일하지 않은경우 "FAIL"
